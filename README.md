\# 🔍 Riventa



\*\*Plataforma de busca inteligente estilo Perplexity\*\*



Uma ferramenta de busca avançada que combina múltiplas fontes de informação para fornecer respostas precisas e contextuais.



\## ✨ Características



\- 🌐 Busca em múltiplas fontes simultaneamente

\- 🤖 Integração com APIs de IA (OpenAI, Anthropic)

\- ⚡ Interface moderna e responsiva

\- 📊 Sistema de cache inteligente

\- 🔍 Resultados contextualizados e sumarizados



\## 🛠️ Tecnologias



\*\*Frontend:\*\*

\- React.js

\- HTML5/CSS3

\- JavaScript moderno



\*\*Backend:\*\*

\- Node.js

\- Express.js

\- APIs REST



\*\*Integrações:\*\*

\- OpenAI API

\- Anthropic Claude API

\- Serviços de busca web



\## 🚀 Instalação



\### Pré-requisitos

\- Node.js (v16 ou superior)

\- npm ou yarn

\- Chaves de API (OpenAI, Anthropic)
